Welcome to SafeGPT's documentation!
===================================

**SafeGPT** helps users to retain their privacy while using any LLM APIs.Check out the :doc:`intro` section to understand why you need it.

It is available on AWS Marketplace <https://aws/marketplace/container/SafeGPT/>`_
and offers a *simple* and *intuitive* API.

Check out the :doc:`usage` section for further information, including
how to :ref:`installation` the project.

.. note::

   This project is under active development.
